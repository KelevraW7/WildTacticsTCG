# README #

# Fully-featured solution for all of your COIN needs! #
Compatible with the Unity Game Engine, this asset provides a full COIN implementation suitable for prototypes, games, or other experiences.

# Flick into action with a drag & drop setup! #
This COIN is designed to be drag & drop into your project and should integrate nicely into any existing project. Source code and samples included.

# Easily customisable! #
The asset comes with 6 different heads & tails designs that can be mixed and matched to suit your needs. The COIN is also UV unwrapped and a UV template is provided to make it incredibly simple to create your own textures.

# A full set of sample scenes! #
Can't make heads or tails of how to use the coin? A set of sample scenes is included to show you the various ways to adjust and toss the COIN!

# Quick-start #
- Import the asset into your project and use the prefabs inside `Assets/wobblewares/Coin/Core/Prefabs`.
- For URP support, import the `URP.unitypackage` located inside `Assets/wobblewares/Coin/`.
- Sample scenes are available inside `Assets/wobblewares/Coin/Samples`. For URP sample support, please import the `URP.unitypackage` located inside `Assets/wobblewares/Kit/`.
Please note that you shoud be able to safely delete the Samples folder (and `wobblewares/Kit` folder) completely if not required.
